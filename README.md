# Murshid - Your BAU Engineering Hub

An interactive platform for Al-Balqa Applied University (BAU) engineering students, providing academic roadmaps, course resources, and faculty information.

## 🚀 Features

- **Interactive Roadmap:** Visualize your academic journey and course prerequisites.
- **Academic Vault:** Access and contribute to academic resources.
- **Faculty Directory:** Explore faculties and majors.
- **Modern UI:** Built with React, Tailwind CSS, and Framer Motion for a premium experience.

## 🛠️ Tech Stack

- **Framework:** React with Vite
- **Language:** TypeScript
- **Styling:** Tailwind CSS + Shadcn UI
- **Animations:** Framer Motion
- **Deployment:** Optimized for Vercel and Netlify

## 💻 Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run development server:**
   ```bash
   npm run dev
   ```

3. **Build for production:**
   ```bash
   npm run build
   ```

## 🌐 Deployment

This project is configured for easy deployment on **Vercel** or **Netlify**.
- **Vercel:** Just connect your GitHub repository; the `vercel.json` file handles routing.
- **Netlify:** The `public/_redirects` file ensures client-side routing works perfectly.
